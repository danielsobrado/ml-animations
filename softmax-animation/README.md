# Softmax Animation

A visual demonstration of the Softmax activation function, built with React, Three.js, and Tailwind CSS.

## Overview

This project visualizes how the Softmax function converts a vector of raw logits into a probability distribution. It includes:

- **Animation Panel**: Step-by-step visualization of the Softmax calculation (Logits $\to$ Exponentials $\to$ Sum $\to$ Probabilities).
- **Graph Panel**: Bar chart comparing input logits with output probabilities.
- **Interactive Practice**: Play with logit values to see real-time updates.

## Tech Stack

- **Vite**: Fast build tool and dev server.
- **React**: UI framework.
- **Three.js**: 3D/2D graphics for the animation.
- **GSAP**: Smooth animations.
- **Tailwind CSS**: Styling.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown (usually `http://localhost:5173`).
